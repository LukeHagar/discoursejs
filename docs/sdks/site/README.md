# Site
(*site*)

### Available Operations

* [getSite](#getsite) - Get site info

## getSite

Can be used to fetch all categories and subcategories

### Example Usage

```typescript
import { SDK } from "@lukehagar/discoursejs";

const sdk = new SDK();

async function run() {
  const result = await sdk.site.getSite();

  // Handle the result
  console.log(result)
}

run();
```
